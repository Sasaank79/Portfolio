# Surya Sasaank Yanamandra - Portfolio

A modern, responsive portfolio website built to showcase my work in backend engineering, machine learning systems, and performance optimization.

## Live Site

**[suryasasaank.dev](https://portfolio-seven-zeta-80.vercel.app/)**

## About This Project

This portfolio is designed with a focus on clean aesthetics and smooth user experience. It features:

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- **Smooth Animations** - Subtle transitions that enhance without distracting
- **Performance Optimized** - Fast load times with minimal bundle size
- **Accessibility** - Semantic HTML and proper contrast ratios

## Tech Stack

| Category | Technologies |
|----------|--------------|
| Frontend | React 18, Vite 5 |
| Styling | Tailwind CSS |
| Animations | Framer Motion |
| Deployment | Vercel |
| Analytics | Vercel Analytics |

## Featured Projects

- **Driving Narrator** - Real-time traffic sign detection using OpenVINO (14 FPS on CPU)
- **URL Shortener** - High-scale backend system with Redis caching (56B capacity)
- **Student Success Prediction** - Serverless MLOps pipeline on AWS Lambda
- **Inventory Management** - Full-stack production app with 96% test coverage
- **Facial Emotion Recognition** - Real-time deep learning with PyTorch

## Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
